# To-Do Inteligente

Aplicação web simples para gerenciamento de tarefas com integração de API.

## Funcionalidades
- Adicionar tarefas
- Marcar como concluída
- Exibir frases motivacionais via API
- Interface simples e intuitiva

## Tecnologias
- HTML
- CSS
- JavaScript
- API REST (Quotable)

## Objetivo
Praticar integração com APIs, manipulação de DOM e organização de dados no front-end.

## Como usar
1. Digite uma tarefa
2. Clique em "Adicionar"
3. Clique na tarefa para marcar como concluída

## Melhorias futuras
- Salvar tarefas no localStorage
- Remover tarefas
- Filtros (pendente/concluída)
